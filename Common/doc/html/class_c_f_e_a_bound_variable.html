Detailed Description

Class that adds storage of boundary variables (tractions) to CFEAVariable.

Note
Member variables are allocated only for points marked as "vertex" i.e. on a boundary. A map is constructed so that variables can be referenced by iPoint instead of iVertex.
Author
R. Sanchez.
